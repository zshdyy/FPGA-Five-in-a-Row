# 基于 VHDL 的 FPGA 五子棋游戏 (VHDL Gomoku)

![Language](https://img.shields.io/badge/Language-VHDL-blue.svg)
![Hardware](https://img.shields.io/badge/Hardware-FPGA-green.svg)

## 📌 项目概览
本项目是一个基于 VHDL 硬件描述语言开发的五子棋游戏。通过自顶向下的设计方法，在 FPGA 开发板上实现了完整的五子棋双人对战逻辑，包括高清棋盘显示、精准落子控制以及实时的胜负判断机制。

## 🛠️ 硬件与开发环境
* **主控芯片**: [Xilinx]
* **开发环境**: [Vivado]
* **显示输出**: VGA 接口显示阵列棋盘
* **交互输入**: [ PS/2 鼠标、开发板自带按键]

## 🧩 核心功能模块
1. **VGA 驱动与显示模块**: 生成标准的 VGA 时序，绘制棋盘网格，并根据状态寄存器渲染黑白棋子。
2. **落子控制模块**: 处理外设输入坐标，包含完善的按键消抖逻辑，更新棋盘内部的状态矩阵。
3. **胜负裁判模块 (核心)**: 实时扫描落子点的横、竖、左斜、右斜四个方向，运用状态机判断是否存在“五子连珠”。
4. **游戏流程控制**: 维护游戏的初始状态、对局状态以及胜负结算状态的平滑切换。
